# Markdown编辑器 v2.0

一个界面友好的Web端Markdown编辑器，提供实时预览、丰富的格式化工具和个性化主题定制功能。

基于EasyMDE，使用Html+JavaScript实现，本地化处理，不需要网络，不需要部署，即点即用。

## ✨ 核心功能

### 📝 编辑功能
- **实时预览**：左侧编辑，右侧实时预览渲染效果
- **丰富的格式化工具栏**：
  - 基础格式化：粗体、斜体、删除线
  - 结构元素：标题、引用
  - 列表功能：无序列表、有序列表、任务列表
  - 媒体元素：链接、图片
  - 表格和代码块
- **快捷键支持**：常用操作支持键盘快捷键

### 🎨 主题定制
- **8种内置预览主题**：
  - 默认、墨黑、姹紫、金钱蓝
  - 绿意、红绯、兰青、科技蓝
- **自定义主题**：内置CSS编辑器，支持完全自定义样式
- **深色/浅色主题切换**：一键切换编辑器整体主题

### 💾 数据管理
- **自动保存**：每3秒自动保存到本地存储
- **手动保存**：支持手动保存操作
- **文件导入**：支持导入.md格式文件
- **文件导出**：
  - 导出为.md格式
  - 导出为.html格式
  - 导出为.pdf格式

### 📊 状态监控
- **字数统计**：实时显示文档字数
- **行数统计**：显示当前文档行数
- **光标位置**：实时显示光标所在行和列
- **保存状态**：显示文档是否已保存

## 🛠 技术栈

### 前端技术
- **EasyMDE**：Markdown编辑器核心库
- **HTML5**：页面结构和语义化标记
- **CSS3**：样式设计和响应式布局
- **JavaScript (ES6+)**：核心功能实现
